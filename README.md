# Home Service Booking Platform

## Project Overview
A web application developed using Flask that enables users to book home services such as plumbing, electrical work, painting, carpentry, AC servicing, and cleaning. The platform includes separate dashboards for users, workers, and administrators.

## Features
- User registration and login
- Worker registration and login
- Admin login and dashboard
- Service booking
- Booking status tracking
- Worker verification
- Feedback system
- Booking history

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite
- Jinja2
- Git
- GitHub

## How to Run
1. Clone the repository.
2. Install Flask.
3. Import the database.
4. Run:
   python app.py
